## Run the API local
1. Install the dependencies:      
[Node.js](https://nodejs.org/en/)

2. Clone the project and install the dependencies:
```

git clone https://github.com/RafaelGoulartB/Ecoleta.git
cd Ecoleta/
cd server/
npm install

```

3. Run

Developement Mode
```
npm run dev
```
